Indicates multiple options for the resource from which the client may choose (via [agent-driven content negotiation](https://en.wikipedia.org/wiki/Content_negotiation#Agent-driven)). For example, this code could be used to present multiple video format options, to list files with different [filename extensions](https://en.wikipedia.org/wiki/Filename_extension), or to suggest [word-sense disambiguation](https://en.wikipedia.org/wiki/Word-sense_disambiguation).<br /><br />[<-- Back](../../)
